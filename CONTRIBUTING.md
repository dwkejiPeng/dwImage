# Contributing to dwImage

Thanks for your interest in contributing.

## Before You Start

- Please open an issue before starting larger changes
- Keep PRs focused on one topic
- Prefer small, reviewable commits

## Development

### Setup

```bash
pip install -r requirements.txt
python app.py
```

### Guidelines

- Keep Windows compatibility intact
- Avoid breaking saved local settings/data
- Prefer incremental UI changes over large rewrites
- Keep the app usable at non-fullscreen window sizes

## Pull Requests

Please include:

- What changed
- Why it changed
- How it was tested
- Screenshots for UI changes when relevant

## Reporting Bugs

When reporting a bug, try to include:

- Windows version
- Python version
- Steps to reproduce
- Expected result
- Actual result
- Error logs or traceback if available
