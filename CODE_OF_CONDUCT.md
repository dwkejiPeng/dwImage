# Code of Conduct

## Our Pledge

We want this project to be a welcoming and respectful space for everyone.

## Expected Behavior

- Be respectful and constructive
- Assume good intent
- Focus feedback on the work, not the person
- Be patient with contributors at different experience levels

## Unacceptable Behavior

- Harassment or personal attacks
- Discriminatory or hateful language
- Trolling, intimidation, or deliberate disruption

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this code of conduct.
