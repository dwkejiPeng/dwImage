# dwImage v0.1.0

`dwImage` 的首个公开版本，重点是把 Windows 本地图像生成工作流先完整跑通。

## 本版本包含

- 文生图
- 参考图生成 / 编辑
- 多提示词批量生成
- 多参考图批量生成
- 同提示词与同配置下按参考图逐张生成
- 窗口任意位置拖拽图片导入
- 粘贴图片 / 图片文件
- 本地历史记录、收藏夹、请求日志
- 多 API Profile 配置
- Prompt Optimization 配置

## 适合谁

- 需要在 Windows 本地高频出图的人
- 需要批量处理多个提示词或多张参考图的人
- 希望把 API 配置、历史记录和生成结果留在本地管理的人

## 已知情况

- 当前版本仅支持 Windows
- 仍会继续优化布局、状态反馈和打包发布体验

## 后续方向

- 更成熟的窗口布局和视觉细节
- 更顺手的多 API Profile 管理
- 更稳定的批量任务反馈和失败处理
- 更完整的发布包与安装体验
