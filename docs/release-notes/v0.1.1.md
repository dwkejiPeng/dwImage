# dwImage v0.1.1

`v0.1.1` 是一次面向交互体验和发行包一致性的补丁更新。

## 本次修复

- 修复生成完成后自动跳转到“历史”页面的问题
- 修复点击开始生成后，右侧状态区仍显示“等待开始”的问题
- 重新构建 Windows 发行版，确保发布包与最新源码一致

## 适合更新的人

- 已经在使用 `v0.1.0` 的 Windows 用户
- 对生成流程状态反馈比较敏感的用户
- 直接使用 GitHub Release 下载 `exe` 版本的用户

## 说明

- 当前版本仍然仅支持 Windows
- 本次更新不改变核心工作流，重点是修复交互细节和发布包同步问题
